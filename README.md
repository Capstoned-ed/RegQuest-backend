📌 Overview

RegQuest is a web-based credential request and real-time tracking system designed for the Registrar’s Office of the University of Science and Technology of Southern Philippines.

This backend system is developed using Django and Django REST Framework (DRF) to support both web and mobile interfaces. It enables students to request academic documents online while allowing registrar staff to efficiently manage requests and monitor workloads.

The system also integrates a Decision Support System (DSS) to forecast daily workloads based on historical request data.
